Feal insignificant? Don't

UC Irvine 的 《Introductory Cosmology W15》从今年春节开始就已经订阅了，但是拖延症作祟，上了一两课之后就束之高阁。

到了八九月换工作休年假之际，才突然想起来 iTunes U 中订阅了这么一门课程，然后趁着中午吃饭、中途等车等时候继续开始看。

这么一门针对大一新生的课程，类似于自己专业的计算机科学导论，完全没有任何艰深的内容，上课的老师 Dr. James Bullock 从两三千年前的古希腊人讨论世界的基本规律开始，用了二十几节课的时间，讲到相对论、大爆炸理论等的方方面面。其中还有若干篇幅的名人轶事，比如众所周知的牛顿爵士的童年往事，爱因斯坦的成长史，还有第谷的鼻子，加州富豪 Lick 的发家史。

对于我这么一个个体，这门课的影响类似于《人类简史》，他们都没有很繁琐的细节，但是通过跨度够大的全局观给了我一个新的思考方向。

比如大爆炸之处，宇宙中只有简单的氢和氦，如此简单的粒子聚集在一起形成最初的一批恒星，通过数亿年记的时间，恒星之中的氢和氦慢慢聚变成更大的原子，质量足够大的恒星足以使聚变的过程进展到合成铁原子，之后最后一批燃料燃烧殆尽，成为超新星爆炸，把合成的各种原子抛洒到星系空间。之后，新一轮的造星过程中，各种合成的原子在重力作用下重新聚集，合成了地球这样的岩石星球，然后又在机缘巧合之下，这些原子又组成了更复杂的形态——生命。我们平时所说的各种金属，各种材料，各种微量元素，还有我们人类这样的各类生命体，在相遇之前，其中的每一个原子都可能已经经历过了数十亿年记的星际旅行。

某一课中，提到银河系这样普通的星系中，估计有千亿级的恒星，而在人类可见的宇宙中，又有百亿级如银河系这样的星系。我们在的地球，完全是银河系千亿恒星中某颗恒星不经意间孕育出的个体。讲到这里，教授问大家是不是觉得自己很渺小（feel insignificant）。但是其实不用这么觉得，因为我们自己是迄今为止宇宙中唯一一个发现了这些事实的群体。虽然可能宇宙中其它地方的智慧生物会提出反对意见，但是至少目前我们还没收到他们的信息。

整门课历史跨度一百四十亿年，涉及到人类文明的跨度也有两三千年，从百年记的跨度回顾历史，能看到人类文明动态的发展，而不是作为一个不过百的人类实例看到的整个世界呈现出的虚幻的静态。

Relax and play

刷牙的时候想到

每隔两三年，感觉自己就会做一次选择题：初中毕业的时候在想到底是继续读高中还是去做电子竞技，高中毕业的时候在想没考上一本的话是找个二本学校上了还是继续复读，大学毕业的时候在想自己是找工作还是出国之类。

初中过了中二之后发现自己天性愚钝，还没到做电子竞技的水平，于是只能放下鼠标继续中考，高考战战兢兢点开查分网站的时候发现自己竟然考上一本了，于是问题不成立，大学毕业的时候发现家里经济水平有限，所以只能开始工作。

脑海中出现的是一棵从小学开始，每隔两三年就分叉一次的二叉树，每次自己做了一个选择，就等于自己生生从人生可能性的二叉树上砍掉了一半的分支。

如果我是一两年前想起这个问题，估计还会给每个节点加上不同的权重：读书高过玩游戏、工作大于出国……

之后看到《How Will You Measure Your Life》，其中作者谈到自己衡量自己人生的标准

I realized that God, in contrast to us, does not need the tools of statisticians or accoutants. So far as I know, He has no organization charts. There is no need to aggregate anything beyond the level of an individual person in order to comprehend completely what is going on among humankind. His only measure of achievement is the individual.

Somehow , after all of this. I came to understand that while many of us might default to measuring our lives by summary statistics, such as number fo people presided over, number of awards, or dollars accumulated in a bank, and so on, the only metrics that will truly matter to my life are the individuals whom I have been able to help, one by one, to become better people.

虽然自己不相信世界上有 root 权限的上帝存在，但是端着 Kindle 看完这段之后，自己内心豁然开朗：所有的这些选择其实都没有具体的是非对错，只是选择而已。

想起《Person of Interest》中 Finch 教他的机器下象棋的时候告诉他，就算你走错一步，也会有无限的可能来修正。

So you should simply relax and play

Android HTTP 摘要: URI 与 URL

一个句话总结：URI 是 URL 的超集

URI (Uniform Resource Identifier)

URI 的定义：scheme-name : hierachical-part ? query # fragment

hierarchical-part 可以分为两类：

  • 以 // 开头，包括 authority 和 path 两部分, 对应的 URI 在 Android 文档中称之为 Hierachical URI
  • 不以 // 开头，只包括 path, 对应的 URI 在 Android 文档中称之为 Opaque URI

scheme-name 其实也和域名一样需要在 IANA 注册。注册的 URI scheme 中有 tel, mailto 等常见的类型，也有things, skype 这样和具体 App 关联的 scheme。

URI 还可以细分成 Absolute URI 和 Relative URI, Relative URI 不包含 scheme-name。

例子：

  • http://www.miui.com: Absolute, Hierarchical
  • mailto:bd-huangye@xiaomi.com: Absolute, Opaque
  • tmp.txt: Relative

URL (Uniform Resource Locator)

URL 的定义：scheme://user:password@domain:port/path?query#fragment。

URL 相比 URI 增加了从何处获取资源的位置信息，这也是其中 Locator 的本意。

http://miui.com 是 URL，而 miui.com 就不是了，因为没有指定 scheme，mailto:hr@xiaomi.com 也不是 URL，因为在其中并不包括具体的资源位置信息，但它是一个合法的 URI。

URN (Uniform Resource Name)

除 URL，URI 中还包括 URN，一种通过命名空间来表示资源（而非位置）的表示法。
URN 的定义：urn: namespance-identifier : namespace-specific-string。

从维基百科拿来的例子：
urn:isbn:0451450523: 《The Last Unicorn》这本书的资源名称
